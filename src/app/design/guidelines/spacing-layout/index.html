SKY UX provides two spacing approaches that can be used together when laying out content: fluid grid and spacing classes. In general fluid grid works well when laying out containers and sections within containers, and the spacing classes work well when laying out content within a section.

Pages

You should use one or two column fluid grid layouts when placing containers on a page. A common example of a one-column layout is a list page – the list takes up the full width. A common example of a two-column layout is a record page – there are two columns of tiles.

Usage
The light separator is a very light gray, single-pixel line that subdivides content within a container. Do not use it at the edge of a container. Only use it when it is adjacent to white space on both sides. SKY UX uses this style for:
  • Subsection dividers within containers such as tiles
  • Top boundaries for action bars on modals
Code
@include sky-border(light)
Styling
  • 1px solid #e2e3e4
  • When this line style subdivides content within a container, use a 15px margin on both ends.
